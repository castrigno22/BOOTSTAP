Bootstrap utilizza un valore predefinito di 1rem
Il font predefinito che utlizza bootstrap è un "native font stack"
Il margine superiore di un paragrafo è 0, il margine inferiore invece è di 1

Classi:

.text_muted :
Viene utilizzato per far "perdere significato" al testo
.text-primary :
Viene ulizzato per dare importanza al testo, assegnadgli il colore blu
.text-success :
Viene ulizzato per dare un impressione di "andato a buon fine", assegnadogli il colore verde
.text-info :
Viene ulizzato per dare un'impressione di "informazione", assegnadogli il colore celeste
.text-warning :
Viene ulizzato per dare un'impressione di "avvertenza", assegnadogli il colore giallo
.text-danger :
Viene ulizzato per dare un'impressione di "pericolo", assegnadogli il colore rosso
.text-secondary :
Viene ulizzato per testo che deve essere in secondo piano, assegnadogli il colore #6c757d
.text-white :
Viene ulizzato per evidenziare un testo, assegnadogli uno sfondo di colore nero
.text-dark :
Viene ulizzato per un testo normale, assegnadogli il colore nero
.text-body :
Viene ulizzato per impostare il colore di default nero, spesso è il nero
.text-light :
Viene ulizzato per un testo normale, assegnadogli il colore nero
